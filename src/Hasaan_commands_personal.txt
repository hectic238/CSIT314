npm - v

** create folder for frontend **

** create react app **
npx create-react-app event_management_system

** go to event_management_system **
cd event_management_system

** start **
npm start



** create folder for backend **

** new project **
npm init -y

** make a .js file for server **
** add **
**
const express = require('express');
const cors = require('cors');

const app = express();
app.use(cors());
app.use(express.json());

app.get('/', (req, res) => {
    res.send('Hello from Node.js backend!');
});

app.listen(5000, () => {
    console.log('Server running on http://localhost:5000');
});
